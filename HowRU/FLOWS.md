### Invite Link Acceptance Flow

When a supporter clicks an invite link:
1. Deep link contains CircleLink ID or invite token
2. App opens to accept screen showing checker's name
3. New user: completes OTP auth, creates User with isChecker=false
4. Existing user: just accepts, CircleLink.acceptedAt set
5. CircleLink permissions come from checker's defaults
